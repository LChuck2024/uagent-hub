# uAgent Hub

This contains everything you need to run uAgent Hub locally.

View your app in AI Studio: https://ai.studio/apps/4f705941-1dc5-4f60-b125-fda5565b4e97

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy [.env.example](.env.example) to `.env` and set `GEMINI_API_KEY` to your Gemini API key
3. Run the app:
   `npm run dev`
